# Secret Spell

A full-screen magic spell page for kids. Type the secret word, cast the spell, and unlock the prize video!

## Live site

**https://treasure-hunt-kohl.vercel.app**

## Repository

**https://github.com/GS-MarkBailey/treasure-hunt**

## Customize

Edit `app.js`:

- **`MAGIC_WORD`** — The secret spell (default: `abracadabra`)
- **`TREASURE_LINK`** — Prize URL (YouTube opens in the YouTube app on mobile)

## Features

- Full-screen spell casting UI
- Prize animation on success
- YouTube app deep link on phones
- Play again to reset
